PotatoPi
PotatoPi is an RP2040-based microcontroller designed for hand soldering. It follows the same pinout as the Raspberry Pi Pico and is fully feature-compatible.

It includes:

16MB of onboard flash memory
A convenient reset button for easy flashing
Custom PotatoPi artwork on both sides of the PCB
