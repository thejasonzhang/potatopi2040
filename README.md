PotatoPi is a RP2040 based microcontroller that will be hand-soldered. The PotatoPi follows the pinout of the Pi Pico and has the same features. It has 16mb of flash memory and has a handy reset button to flash the memory.
Please enjoy the custom potato pi logos and artwork on both sides of the board.
